# Listr
A list app for Shareworks, to access ruby knowledge. This is a time-boxed exercise with 4 issues to replicate the issues that may arise day to day.  You have *4 hours* to resolve as many as these issues as you can. 

Select 1 issue to work on at a time. Please read each issue carefully as their requirements are sometimes not as simple as they suggest. If you are stuck or donâ€™t understand the requirements, it is your responsibility to reach out  for clarification.
